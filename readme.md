### Fix GTA-SA

This is a simple script that sets up GTA San Andreas on Linux

I'm currently updating the script so don't run it yet(unless you know what you're doing)

## Prerequisites

- wget
- jq
- 7z
- Dowgraded version of San Andreas(v1.0)
